ATG context sequences for orthologs of selected genes

Search on fungidb (v38) saved under strategy "Sc eEF1A Orthologs".
All Fungidb orthologs of Saccharomyces cerevisiae eEF1A, translation elongation factor 1a
Step 1: search by S. cerevisiae S288C gene name e.g. "YPR080W".
OR if Scer has 2+ paralogs, search under C. neoformans H99 gene name e.g. CNAG_05900
Step 2: expand by orthology to all annotated fungi, using OrthoMCL
Step 3: download genomic translation start ATG codon context from -12 to +11
